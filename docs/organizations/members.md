# Organization Members

## Membership Types

Members are organized in multiple Membership Types. Every Organization has an Admin Member Type and a Default Member Type.

Any Member can have multiple Types at the same time.

### Permissions

Organization Permissions manage how a User sees the Organization Page and with which Features he can interact.

Main Permission Types are:

* Managing Organization Details
* Managing Members
* Managing Shift Types
* Managing Shift Members

## Adding Members/groups

Members can be added on the Organization Administration Page.

A Member can either be a Group that was created from an external data source or a user of the platform.
For Privacy reasons the username is entered manually.

## Removing Members/Groups

Members and Groups can be removed on the Organization Administration Page using the "*X*" Button next to their name. This will remove the Member or Group from the Organization but not delete the User or Group itself.

## Managing Permissions

Permissions can be edited through the corresponding Membership type on the Organization Administration Page. You can select which Permissions a Membership Type should have and all Members with this Type will inherit these Permissions. This way you can easily manage Permissions for multiple Members at once.
