# Introduction to Shiftings

## What is Shiftings?

Shiftings is a Tool for long Term Shift Management in Organizations. It allows you to create and manage Shifts for your Organization and keep track of who is doing what and when. Shiftings is designed to be flexible and adaptable to the needs of different organizations, making it a valuable tool for managing volunteer work, staff scheduling, or any other type of shift-based work.

## Key Features

* Join Shifts in your Organization
* Import Members from LDAP or OIDC Datasources
* Member level permission management
* Day and Month Calendar Overview
* Create Shifts by Template
* Create Recurring Shifts based on multiple parameters
* Shift ICS File export
